## CSS box-modal
* Accordidng to CSS, all the HTML elements are just a rectangular box.
* These boxes contain five important properties, they are:
	* width
	* height
	* padding
	* border
	* margin
* Whenever an HTML element get created, CSS will create a box modal for the HTML element.
* In CSS, total width and height is calculated by adding the border and padding to the width and height of the element.
* Use the chrome developer tools box-model represenation for understanding about the box-model of an HTML element.
